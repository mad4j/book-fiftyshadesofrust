# Fifty Shades of Rust

Rust funny recipes

## About

This is an mdBook project containing various Rust recipes and examples.

## Building the Book

To build the book locally, you need to have [mdBook](https://rust-lang.github.io/mdBook/) installed:

```bash
cargo install mdbook
```

Then you can build and serve the book:

```bash
# Build the book
mdbook build

# Serve the book locally at http://localhost:3000
mdbook serve
```

## GitHub Pages

The book is automatically built and deployed to GitHub Pages when changes are pushed to the `main` branch. The deployed book can be accessed at the GitHub Pages URL for this repository.
